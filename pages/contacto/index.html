Preguntas Frecuentes

Encuentra respuestas a las consultas más comunes sobre nuestro proyecto.

¿Cómo puedo colaborar con el proyecto?

Puedes contactarnos a través del formulario de arriba o visitando nuestros portafolios individuales para conocer más sobre cada integrante del equipo.

¿El sitio web está optimizado para móviles?

Sí, el sitio web fue desarrollado con diseño responsivo, garantizando una experiencia óptima en dispositivos móviles, tablets y desktop.

¿Qué tecnologías utilizaron?

Utilizamos HTML5 semántico, CSS3 con Grid y Flexbox, JavaScript vanilla, y Figma para el diseño UI/UX. Todo desarrollado siguiendo las mejores prácticas de desarrollo web.

¿Puedo usar este proyecto como referencia?

Este es un proyecto académico desarrollado para la materia Diseño y Desarrollo Web de UADE. Si deseas usarlo como referencia, por favor menciona la fuente.

¿El proyecto incluye funcionalidades backend?

Este proyecto se enfoca en el frontend. Las funcionalidades como formularios y chat son simuladas con JavaScript para fines demostrativos.

¿Cómo fue la organización del equipo?

Cada integrante se especializó en un área: UX/UI Design, Maquetado Web, Programación Frontend y Comunicación Digital, trabajando colaborativamente en todas las etapas.

¿El sitio es accesible?

Sí, implementamos etiquetas semánticas, atributos ARIA, navegación por teclado y contrastes adecuados siguiendo las pautas WCAG.

¿Puedo ver el código fuente?

El proyecto está disponible públicamente. Puedes revisar el código y la documentación técnica en la sección de avances del proyecto.
